Bill Bliss: Modern E-Wallet App
Project Overview
Bill Bliss is a modern Android e-wallet application designed to simplify personal finance management,
focusing on savings, spending tracking, and a dynamic rewards system. The app aims to gamify financial goals,
making it easier and more engaging for users to manage their money and earn rewards for positive financial habits.
Key Features
1. Account & Security
- Google Sign-In: Seamless and secure authentication using the Google Identity platform.
- User Profile: Dedicated screen for viewing and managing user information (email, language, etc.).
2. Core E-Wallet Functionality
- Card Management: Allows users to view, manage, and add new cards to their e-wallet.
- Transaction Tracking: Dedicated functionality for logging and viewing successful transactions and tracking expenses (linked to the missing TransactionsActivity).
3. Rewards and Motivation
The app features a three-tiered rewards system designed to encourage consistent financial behavior:
- Spending Rewards: Rewards users for milestones related to tracking their expenses (e.g., "First Expense Logged").
- Savings Rewards: Rewards users for achieving savings goals and streaks (e.g., "First Deposit," "1 Month Savings Streak," "R1000 Saved").
- Milestone Rewards: Rewards for significant, overall achievements (e.g., "Saved R1000 Total").
4. Settings and Support
- User Settings: Dedicated screen for managing app settings.
- Localization: Supports multiple languages, specifically including Afrikaans (af) and Zulu (zu), to cater to a diverse user base.
- Contact & FAQs: Provides screens for Contact Us and Frequently Asked Questions (FAQActivity).
Tech Stack
- Language: Kotlin
- Platform: Android (using AndroidX)
- Authentication: Google Play Services Auth (com.google.android.gms:play-services-auth)
